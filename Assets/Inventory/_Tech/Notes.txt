NOMENCLATURE

- Hierarchy : Panel / Holder / Items

VOCABULAIRE
- Item : element of an inventory (weapon, fruit, pile of wood, etc.)
- Slot : UI element shows an item
- Container : contains items
- Displayer : displays container's items on its slots
